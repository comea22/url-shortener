# 縮網址網站 Brief Baby
Brief Baby 是一款簡單易用的縮網址工具，  不僅讓冗長的連結瞬間變身輕巧短網址，  
也關注使用者體驗與安全性，  讓分享更方便、更安心。

## 功能特色
*  快速縮短長網址
*  客製化密碼設置
*  簡潔易用的介面
